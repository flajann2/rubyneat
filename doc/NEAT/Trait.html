Overview

Traits

A Trait is a group of parameters that can be expressed

as a group more than one time. Traits save a genetic

algorithm from having to search vast parameter landscapes

on every node. Instead, each node can simply point to a trait and those traits can evolve on their own. (Taken from the C version of NEAT)

Since we wish to allow for different classes of Neurons, this trait idea is super, since all we need to do is have a different trait species for the different node types.
